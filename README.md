# 🔐 Encryption, Decryption & Hashing Portfolio

## 📌 Overview
This repository demonstrates practical implementations of **data security techniques** including:
- Symmetric Encryption (AES)
- Asymmetric Encryption (RSA)
- Hashing Algorithms (SHA-256, MD5)
- File-based encryption/decryption
- Password hashing and verification

The goal is to showcase **secure coding practices** and provide reusable examples for cybersecurity and software engineering projects.

---

## ⚙️ Features
- **Encryption**: Convert plaintext into ciphertext using AES and RSA.
- **Decryption**: Restore original data from ciphertext.
- **Hashing**: Generate fixed-length hash values for integrity checks.
- **Password Security**: Hash and verify user passwords securely.
- **File Handling**: Encrypt and decrypt text files with ease.

---

## 🛠️ Technologies Used
- **Python 3.x**
